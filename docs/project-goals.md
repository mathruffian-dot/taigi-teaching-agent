# 專案目標

本專案的第一階段目標，是上網搜尋臺灣中小學本土語（臺語）的官方教材來源，並將可公開取得、可合法保存的官方教材下載到本專案中，依來源、學習階段、年級、主題、教材類型與使用情境進行分類與整理。

教材整理完成後，專案需進一步分析教材內容，包含課綱對應、詞彙與句型、臺語漢字與臺羅用法、主題分布、教學活動設計、評量題型與可再利用素材。

## 核心目標

本專案未來的核心目標，是讓使用者在專案資料夾中，只需用自然語言描述需求，就能產出臺語各類教材與教學素材。

目前完成度、驗證證據與待補缺口記錄於 `docs/project-goal-status.md`。

自然語言可產出的內容包含：

1. 考卷
2. 簡報
3. 影片
4. 數位互動網站及程式
5. 測驗

## 工作原則

- 優先蒐集官方、公開、可引用的臺灣中小學本土語（臺語）教材。
- 下載與整理時保留教材來源、網址、授權或使用限制、下載日期與檔案版本。
- 教材分類要能支援後續檢索、分析與自然語言生成。
- 正式輸出教材時，臺語漢字、臺羅、課綱依據與適用年級都需要可追溯。
- 圖片、影片、音訊等大型產出不直接大量存入雲端硬碟，依專案設定使用本機輸出目錄。
